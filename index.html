Chapter 1: Values, Types and Operators

Learning to program is about getting down to the level of the computer to make it di things. In this chapter, I learned that a computer only takes in, processes and outputs data. Ultimately, the computer processes astronomically large numbers of bits (two-valued like switches). Data is represented in chunks of bits called values.

"This gives you enough information to use JavaScript as a pocket calulator, but not much more" - Eloquent JavaScript, p.20

JavaScript uses 64 bits to represent a value. Theoretically, this means it can represent 264 numbers or 18 quintilion (18 with 18 zeroes). In practice, bits are needed to store negative numbers and the decimal parts of nonwhole numbers. As such, 64 bits gives somethings closer to 9 quadrillion (9 and 15 zeros).

Number is just one type of value. There are a total of six types of values in the languague:

  • Numbers
  • Strings
  • Booleans
  • Objects
  • Functions
  • Undefined Values

To find out the type of a value, you can use the typeof function. Each type of value has a corresponding set of operators that can be used to manupulate them. For examples, arithmetic operators act on numbers and logical operators act on Booleans. Below is a summary table that captures all six value types of JavaScript and their corresponding operators:
